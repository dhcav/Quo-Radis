## Description
This Shortcut for mobile devices assists with mobile device deployment sessions:
1. Select an existing Jamf Pro user group
2. Scan QR code of device serial
3. Choose specific user from selected user group
4. Choose between outputs:
- Update mobile device record in Jamf Pro via API, or
- Create CSV file for uploading via MUT
5. Display list of completed assignments 

Third party app requirement: 
- PocketMDM - https://apps.apple.com/au/app/pocketmdm/id6751423971

Privileges required:
- Read User Groups
- Read Mobile Devices
- Update Mobile Devices
